About Mariculture

Mariculture refers to the practice of cultivating aquatic organisms, such as fish, shellfish, and plants, in controlled environments like ponds, tanks, or cages. It involves the breeding, rearing, and harvesting of these organisms for commercial, recreational, or conservation purposes, providing a sustainable source of seafood and contributing to food security worldwide.

About Mariculture

We have used the data set available in Kaggle - Mariculture Monitoring System. After Classifying the data, Preprocessin and performed the Exploratory Data Analysis. This data set contains about 1356 records of data in various categories.

How it will works

The project entails creating an IoT and web-based system for real-time Mariculture Monitoring System. It involves developing compact, wireless IoT devices with sensors to monitor Mariculture parameters like Conductivity, Turbidity , Temperature , Ph Values of water. Algorithms and machine learning models are employed for real-time analysis, accurately identifying Potability. A web-based dashboard provides Mariculture professionals and New comers with visualizations of vital potability , alerts for bad conditions. The system is designed to be scalable, reliable, and user-friendly, with integration capabilities for existing Mariculture systems. Testing, validation, documentation, and training are crucial steps for successful deployment in agricultural settings.
